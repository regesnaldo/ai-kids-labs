# AI Kids Labs

An educational platform for teaching kids about artificial intelligence through interactive labs and experiments.

## Getting Started

### Prerequisites

- Git installed on your computer
- A GitHub account

### Initial Setup

If you're creating a new repository from scratch, follow these steps:

#### For Windows Users

1. Create and initialize a new repository:
```bash
cd "C:\Users\YOUR-USERNAME\Desktop"
mkdir ai-kids-labs
cd ai-kids-labs
git init
```

2. Add your files and make your first commit:
```bash
git add .
git commit -m "Initial commit"
```

3. Connect to GitHub and push:
```bash
git remote add origin https://github.com/YOUR-USERNAME/ai-kids-labs.git
git branch -M main
git push -u origin main
```

#### For macOS/Linux Users

1. Create and initialize a new repository:
```bash
cd ~/Desktop
mkdir ai-kids-labs
cd ai-kids-labs
git init
```

2. Add your files and make your first commit:
```bash
git add .
git commit -m "Initial commit"
```

3. Connect to GitHub and push:
```bash
git remote add origin https://github.com/YOUR-USERNAME/ai-kids-labs.git
git branch -M main
git push -u origin main
```

**Note:** Replace `YOUR-USERNAME` with your actual GitHub username. Make sure you've created a repository on GitHub first.

### Cloning an Existing Repository

If the repository is already set up on GitHub, you can clone it:

```bash
git clone https://github.com/YOUR-USERNAME/ai-kids-labs.git
cd ai-kids-labs
```
